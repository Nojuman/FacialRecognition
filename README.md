# FacialRecognition
Facial Expression Recognition with TensorFlow

## Codes
* collect_images.py : collect face images from Bing and Google
* convert_images.py : convert images files(*.jpg, *.png) to dataset
* input_data.py : load the dataset in a form that can be trained and tested
* cnn.py : create Convolutional Neural Network and train them or classify images.
